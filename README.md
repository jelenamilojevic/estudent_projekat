# Estudent projekat

This is a small project implemented using MEAN stack, for a University assignment

## Project setup

1. Clone project 
2. Create new database in mongo called 'estudent_projekat'
3. Setup database (import files from database folder into database)

## How to run project

To run backend:

``` 
cd backend
tsc
npm run serve
```
To run frontend:
```
cd frontend
ng serve --open

```
